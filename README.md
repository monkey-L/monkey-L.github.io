#Hexo个人博客配置
我的个人博客是由 Hexo + NexT主题 搭建生成的，本仓库为已配置好的Hexo和NexT的环境配置，用于发布和部署我的个人博客。  

##使用前需安装以下环境：
###install hexo
npm install hexo-cli -g
###deploy
npm install hexo-deployer-git --save
###search
npm install hexo-generator-search --save
###rss
npm install hexo-generator-feed --save
