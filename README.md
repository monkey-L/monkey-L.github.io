#Hexo个人博客配置
我的个人博客是由 Hexo + NexT主题 搭建生成的，本仓库用于配置和部署我的个人博客。  
仓库的Hexo分支为环境和主题配置，而Master分支则为博客的托管后台（由Hexo分支自动生成和部署）。

##环境安装
1. 安装 [**node.js**](https://nodejs.org/en/)
2. 执行根目录下的脚本 install.sh
